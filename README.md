# 建模比赛文件库

## 文件夹内容

项目根目录/
├─ src/                # 模型程序与源代码
│  ├─ main.py           # 主程序入口
│  ├─ model/            # 选址+路径优化模型
│  └─ utils/            # 工具函数（数据处理、可视化）
├─ data/                # 数据集文件夹
│  ├─ raw/              # 原始数据
│  └─processed/        # 预处理后可建模数据
├─ image/               # 论文插图（地图、可视化、结果图）
├─ paper/               # LaTeX 论文源码
│  ├─ main.tex          # 论文主文件
│  └─ reference.bib     # 参考文献
└─ preprocess.py        # 数据预处理脚本
